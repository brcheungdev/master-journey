## ⚪ Quick Notes 快速提示
```
• Host Bits = 32 − Prefix
• Usable Hosts = 2^(Host Bits) − 2
• 常见 /24 = 254 hosts; /30 = 2 hosts (点对点)
• 大于 /8 的前缀用于更大网络（如 A 类地址）
```
